Implementation of "Towards secure e-voting using ethereum blockchain" (link: https://www.researchgate.net/publication/323318041_Towards_Secure_E-Voting_Using_Ethereum_Blockchain).
Files: Ballot.sol, test_data.txt

1. Change the compiler version in remix to the latest one (0.5.14-nightly.2019.12.5+commit.d2e3933d.Emscripten.clang)

2. Open the "test_data2" file.

3. 1. Open the "test_data" file. 

2. Use the provided data in order to test the execution.

3. Copy and paste the data provided for each function to the deployed functiones in remix (inside the Ballot contract under "Deployed Contracts"). 

4. For voting use the registerd accounts (by switching the accounts under Run tab in remix). 

6. To get the execution cost:
	-  From ganache: Switch to Transactions tab and see "GAS USED"
